HTML DOCUMENTS

All HTML documents must start with a document type declaration: "!DOCTYPE html".

The HTML document itself begins with "html" and ends with "/html".
    
]The visible part of the HTML document is between "body" and "/body"

The " !DOCTYPE " Declaration

  1. The " !DOCTYPE " declaration represents the document type, and helps browsers to display web pages correctly.
  2. It must only appear once, at the top of the page (before any HTML tags).
  3. The " !DOCTYPE " declaration is not case sensitive.

HTML Headings

  • HTML headings are defined with the " h1 " to " h6 " tags
  • " h1 " defines the most important heading. " h6 " defines the least important heading:
example:-- "h1" "/h1"

           "h2" "/h2"

How to view HTML Source

Have you ever seen a Web page and wondered "Hey! How did they do that?

View HTML Source Code

Right-click in an HTML page and select "View Page Source" (in Chrome) or "View Source" (in Edge),
or similar in other browsers. This will open a window containing the HTML source code of the page.

Inspect an HTML Element:

Right-click on an element (or a blank area), and choose "Inspect" or "Inspect Element" to 
see what elements are made up of (you will see both the HTML and the CSS).
You can also edit the HTML or CSS on-the-fly in the Elements or Styles panel that opens.
